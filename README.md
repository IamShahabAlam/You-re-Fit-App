# Youre_fit App

BMI and BMR App (Flutter 2.0)<br/><br/>
Final Year Project For BsCS from SMI University.<br/><br/>
Session (2018 - 2022)

## Features

- Body Mass Index
- Basal Metabolic Rate
- Email Signup
- Phone Signup
- OTP Verification
- Email Login 
- Phone Login 
- 8 Screens
